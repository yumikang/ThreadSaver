ThreadSaver 아이콘 생성 안내

아이콘이 없으면 익스텐션이 로드되지 않을 수 있습니다.

아이콘 생성 방법:

방법 1: create-icons.html 사용
1. create-icons.html 파일을 브라우저로 열기
2. 각 다운로드 링크 클릭하여 저장
3. 이 폴더에 icon16.png, icon48.png, icon128.png 저장

방법 2: 온라인 도구 사용
1. https://www.favicon-generator.org/ 방문
2. 기존 이미지 업로드 또는 새로 만들기
3. 16x16, 48x48, 128x128 크기로 생성
4. 각각 icon16.png, icon48.png, icon128.png로 저장

방법 3: 기존 아이콘 복사
프로젝트의 pixelated-heart-set-three.png를 사용:
1. 이미지 편집기로 열기
2. 크기 조정하여 3개 버전 생성
3. 이 폴더에 저장

임시 방편:
개발 중에는 Chrome이 기본 아이콘을 표시할 수 있습니다.
하지만 정식 배포 전에는 반드시 아이콘을 추가해야 합니다.
